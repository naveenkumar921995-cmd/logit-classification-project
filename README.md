## 📌 Project Overview

This project is a **fully interactive Machine Learning Dashboard** built using **Streamlit**, designed to:

* Compare multiple ML models
* Evaluate performance with different scaling techniques
* Allow real-time predictions
* Enable users to upload their own datasets for predictions

It demonstrates **practical ML engineering + deployment skills**, making it a strong portfolio project.

---

## 🎯 Key Features

### 🧠 Model Comparison

* Logistic Regression
* Support Vector Machine (SVM)
* Random Forest
* K-Nearest Neighbors (KNN)

### ⚙️ Data Scaling Techniques

* StandardScaler
* Normalizer
* No Scaling

---

### 📊 Interactive Dashboard

* Model performance comparison table
* Accuracy visualization chart
* Best model auto-detection 🏆
* Confusion matrix display

---

### 🔮 Prediction Capabilities

* Manual input prediction
* Batch prediction via CSV upload
* Download predicted results

---

### 📁 Real-World Usability

* Upload your own dataset
* Works with dynamic input
* Production-ready deployment

---

## 📊 Model Comparison Output

The dashboard compares all model + scaler combinations and provides:

| Model               | Scaler         | Accuracy    |
| ------------------- | -------------- | ----------- |
| Logistic Regression | StandardScaler | High        |
| SVM                 | StandardScaler | Competitive |
| Random Forest       | No Scaling     | Strong      |
| KNN                 | Normalizer     | Moderate    |

👉 Best model is automatically highlighted in the UI.

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/naveenkumar921995-cmd/logit-classification-project.git
cd logit-classification-project
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run Streamlit App

```bash
streamlit run app/streamlit_app.py
```
